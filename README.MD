## ČESKY
Toto je můj malý projekt na procvičování programování webových stránek. Ukazuje všechny deskové hry a hry na hrdiny, které vlastním ve fyzické podobě.
Deskovky jsou načteny z externího JSONu, který může být upravován a můžu tak přidat více her.

## ENGLISH
This is my little project for practicing website programming. It shows all the board games and role-playing games I own in physical form.
The board games are loaded from an external JSON file, which can be edited so I can add more games.
